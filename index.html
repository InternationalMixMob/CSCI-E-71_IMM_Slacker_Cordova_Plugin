Slacker Plugin

Product Vision

Create Cordova plugins for iOS and Android to enable Slack services within development of any hybrid mobile application.

Key Features of the Slacker Plugin

  • Authorization service
  • Identity Information
  • Selecting Channels
  • Posting Messages
  • Presence Awareness
  • Advance Content sharing via Slash Commands
  • An example Cordova application to demostrate the plugin's features

Try It for Yourself

  1. $ npm install -g cordova
  2. $ cordova create YourAppName
  3. $ cd YourAppName
  4. $ cordova platform add {android|ios}
  5. $ cordova plugin add \
      https://github.com/InternationalMixMob/CSCI-E-71_IMM_Slacker_Cordova_Plugin.git \
      --variable SLACK_CLIENT_ID=xxx \
      --variable SLACK_CLIENT_SECRET=xxx
  6. Inside YourAppName/www/index.html, add:
      <button id='authButton'>Authenticate</button>
    and inside YourAppName/www/js/index.js, at the bottom add:
      document.getElementById('authButton').onclick = function() {
        var success = function() { // do what you want with success };
        Slacker.authenticate(success);
      };
  7. $ cordova run {android|ios}

Node.js is required to use Apache Cordova. You must also have iOS or Android development tools installed

Significant Components

Slacker Client
Native API client to Slack's RestFul Services. It handles communication with Slack and storing or cache state on the device. The goal is to simplfy the orchastration of remote communication, authorization, authentication, presistence, and error handling. Example: Oauth, Token Storage, HTTP Client, Error Handling & Retry, Caching, etc...
 - Client is writen in: Java and Objective-C
 - Client is tested with: Junit and XCTest

Slacker Cordova Plugin
Provides a wrapper to use the Slacker client in hybrid mobile application. It provides Javascript convenience functions that focus on accessing the features of Slack API without needing to deals with the details of how remote communication, authorization, and threading is optimized.
 - Plugin is writen in: Java, Objective-C, Javascript
 - Plugin is tested with: Jasmine

Slacker Reference Application
Created to demostrate the use of the plugin in a typical Apache Cordova application. App contains demos of each feature of the plugin; Login, Listing Channels, Posting Messages, etc.
 - App is writen in: Cordova, HTML5, Javascript, and CSS

Significant Dependencies

Slack API
An Oauth authenticated and authorize Restful API using a JSON request/response payload to integrate and extend the Slack platform.

Apache Cordova Framework
An SDK wrapper that creates app templates for hybrid mobile applications and provides to web container runtime interfaces/plugins which expose native device features. Example, access to device's camera or local storage. Cordova is the evolution of PhoneGap technology and has robust eco-system of plugin vendors and integrators using it to leverage existing enterprise developer HTML skills in mobile development.
